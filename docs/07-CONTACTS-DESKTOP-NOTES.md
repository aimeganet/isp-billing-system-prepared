# ملاحظات Contacts + Desktop

## ما أضيف في هذه المرحلة
- CRUD فعلي لجهات اتصال المشتركين
- دعم رقم أساسي Primary لكل مشترك
- اختيار رقم مناسب للإرسال بحسب القناة: واتساب أو تيليجرام
- تخطي الإرسال آليًا إذا كانت قناة الفاتورة = NONE
- Scaffold فعلي لنسخة Electron
- أوامر تشغيل Desktop محليًا
- أوامر تغليف أولية باستخدام electron-builder

## الملاحظات المهمة
- لم يتم هنا تثبيت `electron` و`electron-builder` داخل البيئة الحالية.
- عند التشغيل على جهازك، نفّذ `npm install` أولًا.
- وضع Desktop الإنتاجي يعتمد على `BUILD_DESKTOP=true` لإخراج Next.js بصيغة `standalone`.
- في نسخة سطح المكتب، قاعدة SQLite المحلية يجب أن تحفظ داخل مسار userData الخاص بالتطبيق حتى تكون مستقلة عن مجلد المشروع.

## ما الذي تحتاج فعله محليًا لبناء نسخة سطح مكتب؟
1. `npm install`
2. `npm run desktop:dev` للتجربة المحلية
3. `npm run desktop:package` لتجهيز الحزمة
4. راقب ناتج مجلد `dist-desktop/`
